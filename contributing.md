# Contributing

Thanks for taking the time to contribute!

## Submitting a Tool

1. Open an issue using the **Tool Submission** template, or open a pull request directly.
2. Make sure the tool is genuinely useful, actively maintained, and on-topic.
3. If you're the maker, please disclose it.

## Pull request format

- Add the entry in alphabetical order within its section.
- Use the format: `* [Tool Name](https://example.com) - One-line description (≤150 chars).`
- Keep descriptions factual; avoid marketing language.
- One entry per PR keeps review fast.

## What we reject

- Affiliate-only entries with no real product
- Abandoned or unmaintained projects
- Off-topic submissions
- Bulk AI-generated entries
